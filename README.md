# Java Control Flow Graph Generator

A Java-based tool for parsing Java source files and extracting method-level control flow information.

## Project Status

Work in progress.

## Goal

The goal of this project is to analyze Java source code and generate Control Flow Graphs (CFGs) for individual methods.

## Technologies

- Java
- Maven
- JavaParser
- Graphviz

## Planned Features

- Parse Java source files
- Extract method declarations
- Detect control-flow structures such as if-statements, loops, and returns
- Build method-level Control Flow Graphs
- Export graph representations for visualization using Graphviz

## Current Focus

Initial project setup and Java source parsing.
